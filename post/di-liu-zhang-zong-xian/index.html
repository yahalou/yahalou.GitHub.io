系统总线结构

突发（猝发）传送：送出一个地址，收到多个地址连续的数据

DMA：Direct Memory Access 直接内存访问

总线的性能指标

  • 总线周期的单位是总线时钟周期

  • 总线周期 -- 总线的工作频率，总线时钟周期 -- 总线的时钟频率

  • 总线带宽 = 总线工作频率 * 总线宽度（速度，10^3）
